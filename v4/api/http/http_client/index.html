http_client

This port provides a common interface for using HTTP clients. Many adapters can be developed to use different technologies.

Its main functionalities are:

  • HTTP, HTTPS and HTTP/2 support

  • Mutual TLS

  • Body encoding/decoding

  • Request/response exchange

  • Form submissions

  • Cookie management

  • File uploading/downloading

Install the Dependency

This module is not meant to be used directly. You should include an Adapter implementing this feature (as http_client_jetty.md) in order to create HTTP clients.
